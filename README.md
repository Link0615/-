
# 五子棋游戏

这是一个基于 HTML、CSS 和 JavaScript 实现的五子棋游戏，包含悔棋和计时功能。

## 功能

- 五子棋游戏
- 悔棋功能（一次撤销两步）
- 计时功能（每步限时 5 秒）
- 开始游戏按钮控制游戏开始

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 游戏逻辑

## 使用方法

1. **克隆仓库**

    ```bash
    git clone https://github.com/你的用户名/你的仓库名.git
    ```

2. **打开项目文件**

    在浏览器中打开 `index.html` 文件。
